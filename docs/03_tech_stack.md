# Tech Stack Specifications

## Core Framework
- **Framework:** React (Vite)
- **Language:** JavaScript (or TypeScript)
- **Styling:** Tailwind CSS (Critical for the specific color codes provided in design system)

## Libraries
- **Icons:** `lucide-react` (Standard icon set)

## Requirements
- **Responsive:** Must work perfectly on Mobile (iPhone 15) and Desktop.
- **Single Page Application:** All content on one scrollable page.
